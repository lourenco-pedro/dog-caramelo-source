Game

Parte principal do jogo. Por onde o jogo é inicializado e executado. Todas as Managers irão ser atualizadas através deste objeto.

Extrutura de execução

Toda execução do jogo vai seguir a seguinte ordem, dentro de Update().

Note: Essa estrutura pode sofrer alteração em sua ordem ou em como vai funcionar.

  1. Atualizar a navegação principal -

    1.1 Atualizar a troca de estado principal de jogo

  1. Atualizar as Managers principais -

    2.1 LobbyManager ( Não se sabe se vai manter esta manager. Por enquanto este Manager deverá ser ignarado no projeto )

    2.2 LevelManager - Atualizar a troca de levels e somente podendo atualizar as demais Managers se o level estiver sido carregado.

    2.3 CharacterManager -

    2.3.1 Atualizar o ControllableCharacter Instanciado no jogo

    2.3.2 Atualizar cada inimigo instanciado, iterando por cada um

    2.4 SpawnManager - Controlar o fluxo de spawn de inimigos

    2.5 BulletsManager - Atualizar todoas as BulletsComponents presentes no jogo

    2.6 DamageManager - Aplicar todos os danos registrados

  1. Atualizar as Managers secuncárias -

    3.1 UIManager - Atualizar toda interface do jogo, apresentando na tela as informações ja atualizadas

Eventos

API

Funções

Publicas

  • ChangeMainState( MainNavigationState toState ): Troca o estado principal do jogo.

Note - O estado sempre é atualizado no próximo frame e nunca no mesmo frame em que se fez o pedido para trocar de estado.

Privadas

  • UpdateMainStateChanging(): Atualiza a troda de estado, se o jogo tiver sido solicitado pela troca de estado, através da função ChangeMainState( MainNavigationState toState ).

Propriedades

Publicas

  • static SimulationSpeed { get }: Retonar a velocidade de simulação do jogo. A velocidade da simulação é definida através da fórmula: _simulationSpeed * Time.deltaTime
  • static CurrentMatch { get }: Retorna o Match principal. A match contém todos os dados da partida atual, como ondas jogadas, número de jogadores, tempo percorrido, etc.
  • MainState { get }: Retorna o estado princial atual do jogo.
